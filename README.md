# The Washboard

A modern website for The Washboard laundromat in Murray, KY.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui components
- React Router

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Project Structure

```
src/
  assets/       - Images and static files
  components/   - Reusable UI components
  data/         - Business data and content
  lib/          - Utility functions
  pages/        - Page components
```
